# Gemini 用 アイコン生成プロンプト（Blokus Solo）

下のブロックを丸ごと Gemini に貼り付けてください。
生成された画像は `blokus-solo/public/icons/` に、指定のファイル名で保存すれば組み込めます。

---

## そのまま貼り付けるプロンプト

```
あなたはゲームUI向けのアイコンデザイナーです。
「Blokus Solo」という、ブロックス（ポリオミノを置く対戦パズル）を1人で練習する
Webアプリ用のアイコンセットを作ってください。

【全体の共通仕様（すべての画像で厳守）】
- ファイル形式：PNG、背景は完全透過（アルファチャンネルあり）
- サイズ：256×256 px の正方形（小さく表示されるアイコン用）
- 余白：被写体の周囲に約10%の余白を均等に取り、中央に配置
- 画風：フラットでややぷっくりした可愛い系。太めで均一な輪郭線。
        ソフトな影で立体感を少しだけ付ける。リアル写真ではなくイラスト。
- 視認性：小さく表示しても何のアイコンか一目で分かるよう、要素を詰め込みすぎず
        シンプル・大きめのモチーフにする。
- 配色：ダーク基調のUIに映える、はっきりした色。
        ブロックスの4色（青 #3B82F6 / 黄 #FACC15 / 赤 #EF4444 / 緑 #22C55E）を基調にする。
- 一貫性：7枚すべて同じ画風・同じ線の太さ・同じライティングで、シリーズとして揃える。
- 文字・透かし・余計な背景は入れない。被写体のみ。

【作ってほしい画像は全部で7種類】

1) app-icon.png — アプリのアイコン（ブラウザのタブ・トップ画面のロゴに使用）
   2×2に並んだ4色（青・黄・赤・緑）の角丸の正方形ブロックを、少し立体的に。
   ブロックスらしさが一目で伝わるシンボル。

2) diff-easy.png — 難易度「弱」（やさしいCPU）
   ニコッと笑った可愛いブロック型ロボット。色は青めで穏やかな表情。初心者向けの安心感。

3) diff-medium.png — 難易度「中」（普通のCPU）
   少し考えている表情のブロック型ロボット。色は黄〜オレンジ。やる気のある雰囲気。

4) diff-hard.png — 難易度「強」（手強いCPU）
   自信ありげで挑戦的な表情のブロック型ロボット。色は赤め。強そうなオーラ。

5) diff-oni.png — 難易度「鬼」（最強CPU）
   小さな角の生えた、不敵に笑う“鬼”モチーフのブロック型ロボット。
   色は濃い紫〜赤で、ラスボス感・絶対に勝てなさそうな威圧感。ただし可愛さも残す。

6) trophy.png — 勝利トロフィー（ゲーム終了・勝利時のお祝い表示に使用）
   金のトロフィーに、4色のブロックの装飾。キラキラした華やかさ。

7) avatar-you.png — プレイヤー（あなた）アイコン
   親しみやすい人型のシンプルなアバター。青を基調に。
   ※6のロボット系と区別がつくデザインにする。

各画像を、上記のファイル名・256×256・透過PNGで、1枚ずつ生成してください。
```

---

## 生成後の使い方（こちらで対応します）

1. 7枚を `blokus-solo/public/icons/` に保存（ファイル名は上記のとおり）
2. 「アイコン置きました」と伝えてください。こちらで以下に差し込みます：
   - `app-icon.png` → favicon ＋ トップ画面ロゴ
   - `diff-*.png` → 難易度選択ボタンの絵（現在は絵文字）
   - `trophy.png` → 勝敗モーダル
   - `avatar-you.png` → プレイヤー表示

> メモ：透過PNGなら現在のダーク背景にそのまま馴染みます。
> もしファイルサイズが大きい場合はこちらで圧縮（縮小）して組み込みます。
